\section{Hướng dẫn sử dụng và giới thiệu:}
\subsection{Giới thiệu các chức năng chính:}
\textbf{Giao diện trò chơi gồm: }\\
- Logo trò chơi: SNAKE\\
- Bảng chức năng: New game, Continue, Highscore, About, Exit\\
\textbf{Game có 2 chế độ: Classic và Modern}\\
- Chế độ Classic: không thể chạm tường\\
- Chế độ Modern: có thể chạm tường và quay ngược lại\\
\\
\textbf{Chức năng của từng menu: }
\begin{itemize}
\item \textbf{Menu New Game: } cho người chơi chọn 1 trong 2 chế độ
\item \textbf{Menu Continue gồm : }tên người chơi và điểm số.
\item \textbf{Menu Highscore: } hiển thị danh sách điểm cao cảu 2 chế độ.
\item \textbf{Menu About: } hiển thị thông tin nhóm sinh viên.
\item \textbf{Menu Exit: } thoát game.
\end{itemize}

\subsection{Các class được sử dụng: }
Chương trình sử dụng class SNAKE 

\subsection{Cấu trúc chính của chương trình}

\begin{itemize}
\item Chương trình có class SNAKE để lưu lại các thông số của con rắn và vẽ con rắn.
\item Vẽ khung viền cho game ngay trong class.
\item Trong class SNAKE chứa các hàm để xử lý trò chơi: Dichuyen, Ve, Taoqua, Vequa, Anqua, Savescore, Savegame, ...
\item Trong file main.cpp có hàm run là menu lựa chọn, hàm play là hoạt động di chuyển của con rắn.
\end{itemize}

\subsection{Hướng dẫn sử dụng: }
Sử dụng 4 phím mũi tên để điều khiển con rắn.\\
Trong lúc chơi game nhấn phím p để pause game và tiếp tục, nhấn F5 để chơi lại từ đầu, nhấn s để lưu game, nhấn ESC để thoát.
